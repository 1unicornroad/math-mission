# Math Mission

An educational iOS game that makes learning multiplication fun! Pilot a spaceship and destroy meteors by solving multiplication problems.

## Features

- **Interactive 3D Gameplay**: Built with SceneKit featuring a spaceship, flying stars, and approaching meteors
- **Customizable Learning**: Choose which multiplication tables to practice (1× through 12×)
- **Difficulty Levels**:
  - **Easy**: 3 answer options, 2 attempts per question
  - **Medium**: 3 answer options, 1 attempt per question
  - **Hard**: 4 answer options, 1 attempt per question
- **Progressive Difficulty**: Meteors approach faster as you answer more questions correctly
- **Performance Tracking**: 
  - Track meteors destroyed
  - See which questions you got on first vs second try
  - Review missed questions to identify areas for improvement
- **Visual Feedback**: 
  - Laser shooting with traveling energy bursts
  - Meteor explosions with realistic fragments
  - Ship damage and explosion effects
  - Dynamic ship banking and positioning

## Requirements

- iOS 13.0+
- Xcode 12.0+
- Swift 5.0+

## Assets

3D models from Kenney's Space Kit (kenney.nl)

## How to Play

1. Select which multiplication tables you want to practice
2. Choose your difficulty level
3. Answer multiplication questions by tapping the correct answer
4. Shoot lasers to destroy meteors with correct answers
5. Avoid getting hit by meteors - you have 3 lives!
6. Review your stats at game over to see what to practice

## Built With

- SceneKit for 3D graphics and animations
- UIKit for user interface
- Swift 5
